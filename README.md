# Biology

一个面向高中生物学习与复习的知识整理仓库。

本项目以 `main.md` 为核心，系统梳理高中生物主要知识点，覆盖细胞、生物代谢、遗传、进化、稳态与调节、生态学以及现代生物技术等内容，适合作为复习提纲、知识速查与 Markdown 笔记参考。

## 内容概览

`main.md` 当前主要包含以下模块：

- 细胞学基础
- 元素和化合物
- 细胞的基本结构与物质运输
- 酶与 ATP
- 细胞呼吸与光合作用
- 细胞分裂、分化、衰老和死亡
- 遗传规律、伴性遗传与分子遗传
- 生物变异与进化
- 内环境、稳态与生命活动调节
- 种群、群落、生态系统与人与环境
- 发酵工程、细胞工程、基因工程与蛋白质工程
- Notes、补充内容与错题整理

## 使用方式

### 在线阅读

直接打开仓库中的 `main.md` 即可浏览全部内容。

### 本地查看

可使用任意 Markdown 编辑器打开，例如：

- VS Code
- Typora
- Obsidian

### 复习建议

1. 按章节通读，先建立整体知识框架
2. 对照教材补充细节与易混点
3. 将高频错题整理到对应章节
4. 考前重点回顾加粗概念与总结内容
